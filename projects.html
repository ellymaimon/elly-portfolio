Forever Animal Shelter Website

Github Repo
  • Languages used: HTML, CSS
  • Contributors: Elly Maimon, Nikki Boyd

This project is a homepage for a fictional animal shelter called "Forever Animal Shelter" and is meant to showcase the box model in CSS. The website features a background image pattern as well as media queries to make the content responsive. In addition, there are images, borders, and the use of divs to create separate sections of content to be custom styled within the page.

Hades All-Inclusive Resort Website

Github Repo
  • Languages used: HTML, CSS
  • Contributors: Elly Maimon, Kelli McCloskey

This project is a fully functional multi-page website for a fictional island resort called "Hades All-Inclusive Resort" and showcases features such as images, links (using anchor tags), a functioning navigation bar, and tables with formatted images.

Wikipedia Homepage Recreation

Github Repo
  • Languages used: HTML, CSS, Bootstrap Framework
  • Contributors: Elly Maimon, Annie Shin

This project is a visual (non-functional) recreation of the Wikipedia homepage. This project features the use of the bootstrap framework to achieve most of the styling. Bootstrap helper classes, wells, panels, and the Grid System are used freely to help achieve the proper resposive functionality while maintaining a visually appealing and accurate recreation.
